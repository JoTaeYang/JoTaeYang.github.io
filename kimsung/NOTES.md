# Notes (teaching scratchpad)

## 학습자 프로필
- 목적: **업무·커리어 준비** — 실무에서 통하는 ML 이해.
- 베이스라인: 머신러닝 **완전 입문(생초보)**. 개발 경험은 있을 수 있음.
- 수학: **거의 안 봄 / 두려움.** 수식 공포 있음 → 그림·비유·직관 먼저, 수식은 이야기로 감싼 뒤 최소한만.
- 언어: 학습자 설명·산출물 **모두 한국어** (교재가 한국어이고 학습자가 한국어 입문자).

## 교육 원칙 (계속 갱신)
- 한 레슨 = 개념 하나 + 즉시 피드백 퀴즈 하나 + 1차 출처(유튜브 원강의) 하나. 짧게.
- 수식을 절대 먼저 던지지 않는다. "왜 이게 필요한가" → 그림 → 말 → (필요하면) 수식.
- 실무/현실 문제로 계속 묶기. "스팸 필터", "시험 점수 예측", "이미지가 고양이냐" 같은 강의 예시 적극 활용.
- 용어는 한글+영어 병기 (실무에서 영어로 만나므로). glossary.html 준수.
