# 0001 — 미션과 출발점

_기록일: 2026-07-15_

## 맥락
학습자가 `kimsung/md/`에 정리된 김성훈 「모두를 위한 머신러닝/딥러닝」 강의(lec/lab 01~06)로
공부하고 싶다고 요청. 커리큘럼 전체는 아니지만 있는 문서 범위 안에서 학습.

## 확정된 사실 (인터뷰로 확인)
- **미션:** 업무·커리어 준비. 실무에서 통하는 ML 이해가 목표.
- **베이스라인:** 머신러닝 완전 입문(생초보).
- **수학:** "거의 안 봄 / 두려움" — 수식 공포. → 그림·비유·직관 우선, 수식은 이야기로 감싼 뒤 최소한만.
- **언어:** 한국어 (교재·학습자 모두 한국어).

## 결정
- 코스 워크스페이스를 `kimsung/` 안에 구성 (blog의 세 번째 코스로 루트 index.html에 카드 등록).
- 공용 에셋(style.css, quiz.js)은 `teach/`에서 복사해 재사용 → 세 코스가 한 몸처럼 보이게.
- 학습 경로: Phase 0(개념) → 선형회귀 → multi-variable → 로지스틱 → 소프트맥스.
  수식이 필수인 경사하강법도 Phase 1에서 "언덕 내려가기" 비유로만 도입, 손계산 미분 안 시킴.

## 다음 세션을 위한 메모 (열린 질문)
- 프로그래밍/Python 경험 수준 미확인 → lab(코드) 레슨 난이도 조절에 필요.
- TensorFlow 실행 환경 유무 미확인 → lab을 "코드 읽기" vs "직접 실행" 중 무엇으로 할지 결정에 필요.
- L1 퀴즈 반응을 보고 회귀/분류 구분이 실제로 몸에 붙었는지 확인 후 Phase 1로.
