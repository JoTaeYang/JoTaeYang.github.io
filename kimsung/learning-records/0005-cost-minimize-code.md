## 다룬 것 (lab 03 기반, read-only 모드)
- L4 수식 `새 W ← 지금 W − α×기울기`를 **코드로 옮기는 것**이 이 레슨의 축. 개념→코드 인터리빙 리듬 유지.
- cost 시각화: placeholder로 W를 −3~5 훑어 cost 찍기(matplotlib) → L4의 밥그릇이 코드로 재현됨(바닥 W=1).
- 방법 ⓐ 수동 3줄: `gradient`(미분=기울기) / `descent`(W−α×기울기) / `update=W.assign(descent)`. 각 줄을 L4 수식과 1:1.
- 방법 ⓑ optimizer 한 줄: `GradientDescentOptimizer(0.1).minimize(cost)` = ⓐ 3줄을 통째 대신. **L3 "조각5=마법버튼"과 명시 콜백** — 그 버튼의 속을 오늘 열었다는 서사.
- 볼록 실증: W=5.0 / W=−3.0 둘 다 1.0 수렴 → L4 "볼록이면 시작점 무관"을 코드로 증명.
- compute_gradients/apply_gradients는 "반복 읽을 때 파볼 지점"으로 콜아웃만(노트: optional, 지금 몰라도 됨).
