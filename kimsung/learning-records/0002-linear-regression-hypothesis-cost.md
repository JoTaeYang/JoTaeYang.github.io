# 0002 — 레슨 2: 선형 회귀의 가설과 cost

_기록일: 2026-07-16_

## 다룬 것 (lec 02 기반)
- 선형 회귀 = 데이터에 가장 잘 맞는 직선 찾기.
- 가설 H(x) = Wx + b — W(기울기), b(y절편)가 선을 정한다.
- cost(비용 함수) = 오차 (H(x)−y)를 제곱해 평균낸 값. 작을수록 좋은 선.
  - 제곱 이유 두 가지: 부호 상쇄 방지, 큰 오차 강조.
- 학습의 목표 = cost(W,b)를 최소화하는 W, b 찾기.
- 숫자 예시: (1,1)(2,2)(3,3)에서 선 A(W=1,b=0) cost=0 vs 선 B(W=0.5,b=0) cost≈1.17로 대비.
