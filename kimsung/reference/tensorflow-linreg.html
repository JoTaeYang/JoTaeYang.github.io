TensorFlow 선형 회귀 골격

랩(lab)에서 반복되는 5+1단계 뼈대. 다음 랩들도 이 위에 얹힙니다. (김성훈 lab 02 기준 · TensorFlow 1.x 스타일)

TensorFlow 버전 주의 이 코드는 TF 1.x 방식입니다(Session·placeholder). 요즘 쓰는 TF 2.x는 문법이 조금 달라요. 지금 목표는 실행이 아니라 읽고 개념과 잇는 것이라 이대로 익혀도 좋습니다.

5+1 단계 뼈대

# 1) 데이터
x_train = [1, 2, 3]
y_train = [1, 2, 3]

# 2) 학습으로 찾을 값 W, b  (텐서플로가 스스로 바꿈)
W = tf.Variable(tf.random_normal([1]), name='weight')
b = tf.Variable(tf.random_normal([1]), name='bias')

# 3) 가설  H(x) = Wx + b
hypothesis = x_train * W + b

# 4) 비용  cost = (예측 - 실제)^2 의 평균
cost = tf.reduce_mean(tf.square(hypothesis - y_train))

# 5) 최소화  (경사하강법)
optimizer = tf.train.GradientDescentOptimizer(learning_rate=0.01)
train = optimizer.minimize(cost)

# +1) 실행: 세션 열고 → 변수 초기화 → 반복 학습
sess = tf.Session()
sess.run(tf.global_variables_initializer())
for step in range(2001):
    sess.run(train)
    if step % 20 == 0:
        print(step, sess.run(cost), sess.run(W), sess.run(b))

개념 ↔ 코드 대응표

코드레슨 개념한 줄 뜻
tf.Variable(...)W, b텐서플로가 학습하며 스스로 바꾸는 값
x_train * W + b가설 H(x)=Wx+b이 선이 내놓는 예측
tf.square(...)오차 제곱부호 없애고 큰 오차에 벌점
tf.reduce_mean(...)평균모든 점의 벌점을 평균 → cost
optimizer.minimize(cost)cost 최소화cost를 줄이도록 W,b를 조금씩 이동
learning_rate=0.01보폭한 번에 얼마나 크게 이동할지 (다음 레슨)
sess.run(train)1회 학습이 줄이 돌 때마다 W,b가 한 걸음 개선

placeholder 변형 (데이터를 나중에 주입)

X = tf.placeholder(tf.float32, shape=[None])   # 값은 나중에
Y = tf.placeholder(tf.float32, shape=[None])
hypothesis = X * W + b
# ... train 실행 시 데이터를 feed_dict로 던져줌:
sess.run([cost, W, b, train], feed_dict={X: [1,2,3], Y: [1,2,3]})
# 학습된 모델로 예측:
sess.run(hypothesis, feed_dict={X: [5]})       # → 약 6.1

shape=[None] = 1차원인데 개수는 자유(3개든 100개든). feed_dict = 그때그때 넣어줄 데이터. _ = "받긴 하지만 안 쓸 값".
