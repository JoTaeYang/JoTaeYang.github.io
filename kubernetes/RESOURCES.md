# Kubernetes Mastery Resources

## Knowledge

- [공식 문서: Kubernetes Documentation — Concepts](https://kubernetes.io/docs/concepts/)
  1차 자료의 기준점. 모든 레슨의 개념 설명은 여기에 근거한다. 용도: 개념 정의, 최신 동작 확인.
- [공식 문서: Kubernetes Tasks](https://kubernetes.io/docs/tasks/)
  실습 절차의 1차 자료. CKA/CKS 시험 중에도 열람 가능한 문서. 용도: 실습 단계, 명령어 확인.
- [CNCF 자격증 공식 페이지: KCNA](https://www.cncf.io/training/certification/kcna/) · [KCSA](https://www.cncf.io/training/certification/kcsa/) · [CKA](https://www.cncf.io/training/certification/cka/) · [CKAD](https://www.cncf.io/training/certification/ckad/) · [CKS](https://www.cncf.io/training/certification/cks/)
  시험 범위(curriculum), 응시 조건, 비용의 유일한 공식 출처. 용도: 커리큘럼 매핑, 시험 정책 확인. CKS는 유효한 CKA 보유가 응시 조건.
- [Kubestronaut 프로그램](https://www.cncf.io/training/kubestronaut/)
  5종 자격증 전부 보유 시 부여되는 CNCF 공식 타이틀. 용도: 최종 목표 기준.
- [killer.sh — CKA/CKAD/CKS 시뮬레이터](https://killer.sh/)
  실제 시험보다 어려운 공식 연계 모의시험. Linux Foundation 시험 등록 시 2세션 포함. 용도: 시험 직전 실전 점검.
- [Killercoda — 무료 인터랙티브 시나리오](https://killercoda.com/killer-shell-cka)
  브라우저에서 바로 실습 가능한 무료 시나리오(CKA/CKAD/CKS). 용도: 레슨별 실습 환경.
- [kind 공식 문서](https://kind.sigs.k8s.io/)
  로컬 멀티노드 클러스터 구축 도구. 용도: 로컬 실습 환경 구성.
- [Book: _The Kubernetes Book_ — Nigel Poulton](https://www.oreilly.com/library/view/the-kubernetes-book/9781098144371/)
  매년 개정되는 입문~중급 교재. 용도: 초반 단계 개념의 서사적 이해.
- [Book: _Kubernetes Up & Running_ (3rd ed.) — Hightower, Burns, Beda 외](https://www.oreilly.com/library/view/kubernetes-up-and/9781098110192/)
  공동 창시자들이 쓴 표준 교재. 용도: 중급 단계 워크로드/운영 개념.
- [Kelsey Hightower: Kubernetes The Hard Way](https://github.com/kelseyhightower/kubernetes-the-hard-way)
  컴포넌트를 손으로 하나씩 조립하는 튜토리얼. 용도: CKA 준비 중 아키텍처 심화(중반 이후에만).

## Wisdom (Communities)

- [Kubernetes 공식 Slack (slack.k8s.io)](https://slack.k8s.io/)
  #kubernetes-users, #cka-exam-prep 등 채널. 용도: 실습 막힘 해소, 시험 준비 정보.
- [r/kubernetes](https://www.reddit.com/r/kubernetes/)
  운영 사례와 자격증 후기(합격 전략, 최신 시험 변경사항)가 활발. 용도: 시험 전략, 최신 동향.
- [discuss.kubernetes.io](https://discuss.kubernetes.io/)
  공식 포럼. 용도: 재현 가능한 기술 질문.
